# 📰 Fake News Detection System

An AI-powered web application that detects whether a news article is **Fake or Real** using Natural Language Processing.

🔗 **Live Demo:** [https://your-app-link.streamlit.app](https://fake-news-detector-a7qdlaaeqy3q299p4oclj8.streamlit.app/)

---

## 🚀 Features

- 🧠 NLP-based classification using TF-IDF
- 📊 Confidence score for predictions
- 🔍 Explainable AI (keyword-based reasoning)
- 🎨 Modern Streamlit UI
- ⚡ Real-time predictions

---

## 🛠 Tech Stack

- Python
- Scikit-learn
- NLTK
- Streamlit

---

## 📌 How It Works

1. Input news article  
2. Text preprocessing (cleaning, stopwords removal)  
3. TF-IDF vectorization  
4. Model prediction (Logistic Regression)  
5. Output:
   - Fake / Real classification  
   - Confidence score  
   - Explanation
